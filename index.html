Política de Privacidade

Última atualização: Julho de 2026

A presente Política de Privacidade explica como o aplicativo OlimpMath gerencia e protege as informações dos usuários. Nosso compromisso é garantir a transparência e a segurança no ambiente educativo.

1. Coleta de Dados

O OlimpMath foi desenvolvido com foco na privacidade. O aplicativo permite o acesso de forma anônima e pública. Não coletamos dados pessoais identificáveis, números de telefone, contatos ou e-mails de estudantes no modo padrão de simulados.

2. Uso de Serviços de Terceiros

Para o correto funcionamento do sistema de simulados e banco de dados, o aplicativo utiliza serviços terceiros confiáveis:

  • Supabase: Utilizado como banco de dados em nuvem para hospedar de forma segura as questões, imagens das provas e registrar o histórico local de acertos/desempenho obtido nos testes.
  • Expo / EAS: Infraestrutura tecnológica utilizada para a compilação, atualização e distribuição do aplicativo.

3. Segurança dos Dados

Implementamos medidas de segurança técnicas para proteger os dados operacionais do aplicativo contra acessos não autorizados ou alterações.

4. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade periodicamente. Recomenda-se que você revise esta página regularmente para verificar quaisquer alterações.

5. Contato

Em caso de dúvidas sobre esta política, entre em contato através do ambiente institucional da escola.
